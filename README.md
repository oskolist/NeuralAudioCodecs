# NeuralAudioCodecs
a small list of available Neural Audio Codecs

- APCodec: [Source code](https://github.com/YangAi520/APCodec), [Demo](https://yangai520.github.io/APCodec/)
- BigCodec: [Source code](https://github.com/Aria-K-Alethia/BigCodec), [Demo](https://aria-k-alethia.github.io/bigcodec-demo/)
- DAC: [Source code](https://github.com/descriptinc/descript-audio-codec), [Demo](https://descript.notion.site/Descript-Audio-Codec-11389fce0ce2419891d6591a68f814d5), [Jax Port](https://github.com/DBraun/DAC-JAX) [html](https://arxiv.org/html/2405.11554v1) [pdf](https://arxiv.org/pdf/2405.11554)
- DualCodec: [Source code](https://github.com/jiaqili3/dualcodec), [Demo](https://dualcodec.github.io), [Huggingface](https://huggingface.co/amphion/dualcodec)
- Encodec: [Source code](https://github.com/facebookresearch/encodec), [Demo](https://ai.honu.io/papers/encodec/samples.html)
- FlowDec: [Source code](https://github.com/facebookresearch/flowdec), [Demo](https://sp-uhh.github.io/FlowDec/)
- FunCodec: [Source code](https://github.com/modelscope/FunCodec), [Demo](https://funcodec.github.io/)
- HILCodec: [Source code](https://github.com/aask1357/hilcodec), [Demo](https://aask1357.github.io/hilcodec/)
- HiFi-Codec: [Source code](https://github.com/yangdongchao/AcademiCodec) [Forked](https://github.com/moewiee/hificodec), [Huggingface](https://huggingface.co/Dongchao/AcademiCodec/tree/main)
- Lyra: [Source code](https://github.com/google/lyra)
- MLow: [Demo](https://engineering.fb.com/2024/06/13/web/mlow-metas-low-bitrate-audio-codec/) [Hardware implementation](https://github.com/vyges/mlow-codec-ip)
- MelCap: [Demo](https://melcapdemo.github.io/Mel_cap_demo/)
- Moshi/Mimi: [Source code](https://github.com/kyutai-labs/moshi), [Demo](https://moshi.chat/), [Huggingface](https://huggingface.co/collections/kyutai/moshi-v01-release-66eaeaf3302bef6bd9ad7acd)
- S-PRESSO: [Demo](https://zineblahrichi.github.io/s-presso/)
- SAC: [Source code](https://github.com/Soul-AILab/SAC), [Demo](https://sac-codec.github.io/)
- SNAC: [Source code](https://github.com/hubertsiuzdak/snac), [Demo](https://hubertsiuzdak.github.io/snac/), [Jax Port](https://github.com/boris-kuz/SNAX)
- SoundStream: [Source code](https://github.com/wesbz/SoundStream) [2](https://github.com/haydenshively/SoundStream)
- SpeechTokenizer: [Source code](https://github.com/ZhangXInFD/SpeechTokenizer), [Demo](https://0nutation.github.io/SpeechTokenizer.github.io/)
- Stable Codec: [Source code](https://github.com/Stability-AI/stable-codec), [Demo](https://stability-ai.github.io/stable-codec-demo/), [Huggingface](https://huggingface.co/stabilityai/stable-codec-speech-16k)
- TF-Codec: [Source code](https://github.com/microsoft/TF-Codec)
- TSAC: [Source code](https://bellard.org/tsac/tsac-2024-04-08.tar.gz), [Demo](https://bellard.org/tsac/)
- UniAudio/LLM-Codec: [Source code](https://github.com/yangdongchao/LLM-Codec)
- UniSRCodec: [Demo](https://wxzyd123.github.io/unisrcodec)
- WavTokenizer: [Source code](https://github.com/jishengpeng/WavTokenizer), [Demo](https://wavtokenizer.github.io/), [Huggingface](https://huggingface.co/novateur/WavTokenizer-large-speech-75token)
